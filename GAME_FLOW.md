# 🥊 Бойцовский Клуб — Игровая инструкция

> Пошаговое описание user flow: что происходит на экране, что делает ведущий (admin),
> что делают зрители (игроки со смартфонов).

---

## Подготовка

1. Запустить сервер: `docker compose up --build -d`
2. Открыть на проекторе/ноутбуке: **`http://localhost:3000/?role=admin`**
3. Раздать зрителям ссылку (по Wi-Fi): **`http://<IP-адрес>:3000/`**
4. Все видят одну и ту же картинку. Управлять может **только admin**.

---

## Сцена 1 — Двери клуба (`landing`)

**Что на экране:**
Ночной вид снаружи клуба. Анимированная дверь (мигает свет).

**Admin:**
Наводит мышку на дверь → появляется текст «Войти в бойцовский клуб?» → кликает.
Дверь открывается (анимация + звук скрипа).

**Зрители:**
Видят экран ожидания: «Ожидание начала игры...»

**→ Переход в сцену 2**

---

## Сцена 2 — Интро: Качер vs Жека (`intro`)

**Что на экране:**
Два персонажа выезжают слева и справа. Между ними — облачко диалога.

**Диалог (туториал):**
1. Качер: *«Ты пидор»*
2. Жека: *«Нет, ты пидор»*
3. Качер: *«Давай драться?»*
4. Жека: *«Давай!»*

**Admin:**
Кликает в любое место экрана → диалог продвигается.
После последней реплики → клик переводит к экрану ставок.

**Зрители:**
Наблюдают диалог.

**→ Переход в сцену 3**

---

## Сцена 3 — Экран ставок (`prefight`)

**Что на экране:**
- Карточки двух бойцов: портреты (`character_sheet.png`) + характеристики
- Полоски сравнения: сила, ловкость, выносливость, интеллект, мудрость, харизма
- Коэффициенты ставок (например: Качер ×1.3 / Жека ×2.8)
- Текущий баланс (в туториале — 0 ₽)
- **VS** по центру

**Admin (внизу экрана — контролы):**
1. Выбрать бойца для ставки (выпадающий список)
2. Ввести сумму ставки
3. Нажать «Ставка!»
4. Нажать большую кнопку **⚔️ БОЙ! ⚔️**

> 💡 В туториальном бою баланс = 0, поэтому ставить необязательно.

**Зрители:**
Видят карточки бойцов и могут оценить, за кого болеть.

**→ Переход в сцену 4**

---

## Сцена 4 — Бой (`fight`)

**Что на экране:**
- Арена (анимированный фон)
- Два бойца: спрайт-анимации idle/attack
- Полоски HP сверху (красная → зелёная)
- HP уменьшаются автоматически (бой идёт на сервере)
- Рывок спрайта + вспышка при ударе
- Всплывающие фразы «БЕЙ!», «ТАК ЕГО!» в месте тапа

**Admin:**
Ничего не нажимает — бой идёт автоматически. Может тапать, как зритель.

**Зрители (ЭТО ГЛАВНЫЙ МОМЕНТ!):**
Тапают по экрану смартфона! 📱
- Тап в **левую** половину → ускоряет атаку **левого** бойца
- Тап в **правую** половину → ускоряет атаку **правого** бойца
- Каждый тап уменьшает интервал следующей атаки на 5% (до −50%)
- На экране появляются фразы: «ДАВАЙ!», «МОЧИ!», «КРАСАВА!»

**Бой заканчивается**, когда HP одного из бойцов падает до 0.

**→ Переход в сцену 5**

---

## Сцена 5 — Результат боя (`postfight`)

### Фаза 1: Fatality
**Что на экране:**
Победитель показывает fatality-анимацию по центру.
Текст **«FATALITY»** красными буквами. Звук добивания.

### Фаза 2: Результат
**Что на экране:**
- Победитель — анимация winner (танцует/радуется)
- Проигравший — анимация beaten (лежит)
- Текст: «🏆 [Имя] ПОБЕДИЛ!»
- Выигрыш/проигрыш ставки (анимированный счётчик)
- Новый баланс

**Admin:**
Кликает → переход к следующему этапу.

**Зрители:**
Радуются или грустят.

**→ Переход в сцену 6 (после туториала) или сцену 2 (следующий бой)**

---

## Сцена 6 — Качер раздаёт деньги (`postintro`)

> ⚠️ Только после туториального боя (раунд 0 → 1).

**Что на экране:**
Качер по центру в idle-анимации. Облачко:
*«Кстати, а я рассказывал, что по ночам сдавал наш офис под хостел? Вот 100к заработал, берите, мне не жалко!»*

**Admin:**
Кликает → запускается анимация начисления +100 000 ₽.
Ещё клик → переход к следующему бою.

**Зрители:**
Видят, как баланс нарастает.

**→ Переход в сцену 3 (экран ставок следующего боя)**

---

## Раунды 2–6 — Основные бои

Цикл повторяется:

```
Intro (диалог двух новых бойцов)
  ↓
PreFight (ставки — теперь с балансом!)
  ↓
Fight (бой + тапы зрителей)
  ↓
PostFight (результат + выплата ставки)
  ↓
Intro следующей пары...
```

Пары выбираются рандомно из оставшихся персонажей (без повторов).
Когда все пары исчерпаны → финальный экран.

> 📌 Сейчас доступны только Качер и Жека. Когда добавятся остальные 10 персонажей,
> будет 5 раундов основных боёв (5 пар из 10).

---

## Сцена 7 — Финал (`win`)

**Что на экране:**
- 🏆 пульсирующий трофей
- Текст: «ВЫ ВЫИГРАЛИ!»
- Итоговый баланс (зелёный если плюс, красный если минус)
- Количество раундов

**Admin:**
Нажимает «Закрыть» → игра сбрасывается к сцене 1 (двери клуба).

---

## Шпаргалка для ведущего (Admin)

| Сцена | Действие Admin |
|-------|---------------|
| Двери | Кликнуть на дверь |
| Интро | Кликать → продвигать диалог → последний клик = «К бою» |
| Ставки | Выбрать бойца, ввести сумму, «Ставка!», потом **⚔️ БОЙ!** |
| Бой | Ничего (можно тапать как зритель) |
| Результат | Кликнуть → следующий раунд |
| Качер 100к | Кликнуть → начисление → кликнуть → далее |
| Финал | «Закрыть» → сброс игры |

## Шпаргалка для зрителей

| Сцена | Действие зрителя |
|-------|-----------------|
| Двери | Ждать |
| Интро | Смотреть |
| Ставки | Смотреть |
| **Бой** | **ТАПАТЬ!** Левая половина = помощь левому, правая = правому |
| Результат | Смотреть |
| Финал | Радоваться |

## Схема переходов

```
landing ──[дверь]──→ intro ──[диалог]──→ prefight ──[БОЙ!]──→ fight
                                                                  │
                                                           HP=0 ──┘
                                                                  │
                                                                  ▼
                                                             postfight
                                                                  │
                                              ┌───────────────────┤
                                              │ раунд 0→1         │ раунд N
                                              ▼                   ▼
                                          postintro           intro (следующая пара)
                                              │                   │
                                              └──→ prefight ◄─────┘
                                                      │
                                              (нет пар осталось)
                                                      │
                                                      ▼
                                                     win ──[закрыть]──→ landing
```
